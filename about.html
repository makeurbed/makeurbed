About

Fantasy houses and dream bedrooms, scroll to explore. This project is an experiment with
direction through 3D space, playing with the in-between of video, still images, and games.

Compatibility

For interactive features, view on desktop and explore by scrolling.
Should work on Google Chrome 9+, Firefox 4+, Opera 15+, Safari 5.1+, Internet Explorer 11.
If you have issues running it, make sure you have JavaScript enabled.
